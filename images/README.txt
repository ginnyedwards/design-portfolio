PUT YOUR IMAGES IN THIS FOLDER
==============================

The site shows striped placeholder boxes wherever an image goes. To replace a
placeholder with a real image, just drop a file into this folder using the exact
name below, then (in the matching .html page) swap the <div class="placeholder">
for an <img>. Each page has a comment showing the line to use, e.g.

    <img src="images/norma-jeanne-1.jpg" alt="Phrenology skull">

Filenames the pages expect:

  home-card-1.jpg      Norma Jeanne thumbnail
  home-card-2.jpg      The Brutalist Jigsaw thumbnail
  home-card-3.jpg      Escape To Cannibalism thumbnail
  home-card-4.jpg      A Story Of Excess thumbnail
  home-hero.jpg        (optional) your real "VIRGINIA EDWARDS" hero, if you'd
                       rather use the photo than the code-generated pattern

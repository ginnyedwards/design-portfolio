Contact Information

Modernism: Living and Breathing in South London is a project that has been taking place since September 2025, and it is a project that I see myself working on indefinitely. The project is based around creating work that keeps Modern architecture alive and work that proves it not to be dead. As of now I have created work such as wearables, embroidered tapestries, drawings and poem books. I aim to constantly add to this collection of work and put it out into the world for people to interact with.
